○ゲーム概要
	
	このゲームは魚をエサで川の上流までたどり着けるように誘導するゲームです。
	あなたは川に棲む神様となり、魚が無事川の上流にたどり着けるよう導く必要があります。
	はたして、何年間魚が生活できるように導くことが出来るでしょうか？
	
○ゲームルール
	魚は川の上流に向かって泳いでいきます。
	しかし、道中で岩・ゴミといった障害物があり、
	魚がこれらに当たってしまうと死んでしまいます。
	
	そこで、川にエサを設置し、魚を誘導することで、魚を岩やゴミから守ることが出来ます。
