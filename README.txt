○ゲーム概要
	
	このゲームは魚をエサで川の上流までたどり着けるように誘導するゲームです。
	あなたは川に棲む神様となり、魚が無事川の上流にたどり着けるよう導く必要があります。
	はたして、何年間魚が生活できるように導くことが出来るでしょうか？
	
○ゲームルール
	魚は川の上流に向かって泳いでいきます。
	しかし、道中で岩・ゴミといった障害物があり、
	魚がこれらに当たってしまうと死んでしまいます。
	
	そこで、川にエサを設置して魚を誘導し、魚を守りましょう。
	
	一年間にやってくる魚の内、5匹以上が死んでしまうとゲームオーバーです。
	もし守り切ることが出来た場合は、年が進み世代交代が起きます。
	川の中で流れる、生命の流れをお楽しみください。
	
	
○操作説明
	ゲーム中の操作はマウスの操作のみで行います。
	（ゲーム中）
	左クリック長押し…押した場所にエサを設置
	※設置したエサは５秒後に消えます。また、３つ以上設置しようとした場合、最初の１つ目が消えます。
